# CFO AI Agent System

A multi-agent system for construction company financial management, driven by LLM reasoning.

## Agents
1. **DocProcessor**: Meticulous accountant for document reading.
2. **FinanceManager**: Strategic finance head for money management.
3. **CFOBrain**: Wise CFO orchestrator.
4. **HumanInterface**: Clear communicator for human interaction.

## Setup
1. Clone the repository.
2. Install dependencies: `pip install -r requirements.txt`
3. Configure environment variables in `.env` (use `.env.example` as a template).
4. Run the system: `python main.py`
